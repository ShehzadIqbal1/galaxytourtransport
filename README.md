# Galaxy Tour & Transport

Marketing website for **Galaxy Tour & Transport** — UAE tours and chauffeur transport with WhatsApp-first booking.

## Quick start

```bash
cp .env.example .env.local
npm install
npm run dev
```

See [`AGENTS.md`](./AGENTS.md) for architecture and [`rules.md`](./rules.md) for engineering rules.

## Scripts

| Command | Purpose |
| --- | --- |
| `npm run dev` | Local development |
| `npm run build` | Production build |
| `npm run lint` | ESLint |
| `npm run typecheck` | `tsc --noEmit` |
| `npm test` | Vitest unit tests |
